Indonesia Keyboard / Papan Ketik Indonesia

To type in Indonesian language. This is a 15-keys keyboard layout, based on the standard E.161 (https://en.wikipedia.org/wiki/E.161) ITU Telecommunication Standardization Sector's Recommendation that defines the arrangement of digits, letters, and symbols on telephone keypads, and then modified to separate the foreign letters 'xq' button, and adding 'ng' and 'ny' buttons. If you're looking for a regular 12-keys keyboard, see my "english-12-keys" keyboard.
Untuk mengetik bahasa Indonesia. Ini adalah papan ketik 15-tombol, berdasarkan standar E.161, kemudian dimodifikasi untuk memisahkan huruf asing 'xq', dan menambahkan 'ng' dan 'ny'.

The physical layout remain unchanged. For touch layout, the main features include:
Untuk papan ketik fisik, tidak ada perubahan. Untuk layar sentuh, fitur-fitur utamanya:

  1. Button rotation/rotasi simbol: Tap button multiple time to get the letter you wanted. For example, tap the "abc" button twice to get the letter "b", and thrice to get "c". The same with the symbols and punctuations (,?!) and (':").
    2. and (/())
  2. (Note on button rotation: Unlike the original numpad keyboard, which has a button delay (for example, press "abc" once to get an "a", then wait a second, press "abc" again, and you'll get "aa") there is no button delay on Keyman (if you did the previous routine, you'll end up with "b" no matter how long the wait, or even if you clicked somewhere else first). So I had to use "zero-width joiner" (on the empty button up left / number 1 location). Basically it's an invisible space, and it tells the keyboard to start the button rotation sequence from the beginning.)
  3. Longpress/tekan lama: if the button has a dot on the top right corner, you can longpress it to access additional numbers or symbols/jika tombol memiliki titik di kanan atas, Anda dapat menekan lama untuk memunculkan angka atau simbol lainnya. If you want to type a number, you can either longpress the corresponding button -- for example "abc" is "2" -- or press the "123" button to get the numeric keypad. Some other buttons also have the longpress keys, such as the punctuations buttons, and the "spacebar" (0, qwerty keyboard, and hide keyboard button).
  4. 'ny' and 'ng' bigraph letters/huruf konsonan ganda 'ny' dan 'ng'
